On a GPU,   
the threads are always   
run as a group of threads.

The GPU kernel is a   
function that runs on GPU.

All the threads in a threadBlock are   
expected to run the same instructions   
and can share memory if specified.

For more information, please refer to:   
[All about GPU Concept](https://dzone.com/articles/all-about-gpu-threads-warps-and-wavefronts)
